classification

Classify document into taxonomy and topics

Classify document content into an appropriate taxonomy and suggest relevant topics.

This endpoint performs a two-stage LLM-based classification:
1. Analyzes content to suggest which taxonomy it belongs to
2. Suggests relevant topics within the selected taxonomy

The classification results can optionally be persisted for historical tracking.
Request Body schema: application/json
required
content
required
string (Content) [ 1 .. 50000 ] characters

Document content to classify

Document Id (string) or Document Id (null) (Document Id)

Optional document ID if classifying existing document

Document Type (string) or Document Type (null) (Document Type)

Type of document (Note, Document, etc.)

max_topics
integer (Max Topics) [ 1 .. 20 ]
Default: 5

Maximum number of topics to suggest

min_confidence
number (Min Confidence) [ 0 .. 1 ]
Default: 0.3

Minimum confidence threshold for suggestions

store_result
boolean (Store Result)
Default: true

Whether to persist the classification result

Taxonomy Hint (string) or Taxonomy Hint (null) (Taxonomy Hint)

Optional taxonomy hint to scope the classification

Responses

Development server

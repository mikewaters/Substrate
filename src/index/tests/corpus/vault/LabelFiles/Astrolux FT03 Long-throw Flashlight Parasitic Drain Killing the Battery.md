# Astrolux FT03 Long-throw Flashlight Parasitic Drain Killing the Battery

Oct 6, 2025

Thought it was fixed; nah, the battery drained again.

Sep 26, 2025

Enabled mechanical lockout with four clicks (which is followed by four blinks to confirm). Disabling uses same procedure.

Aug 3, 2025

### Next actions

1. Based on whatever mode the lamp is in, try the mechanical lockout procedure

2. Disable the LED and see if the drain goes away meaningfully

### Issue

Battery drain to the LED.

> There is a very small drain on the battery while
>
> OFF. This should not be of any concern, but we
>
> recommend removing the cell (or partially
>
> loosening the tail-cap) if the FT03 will not be used
>
> for several months.

### Solution

Many users just keep the battery cap unscrewed slightly. Other option is mechanical lockout using the battery firmware.

# Browser Keystroke Implementation Guide

## Safe Universal Keystrokes

### Navigation Keys

- Tab

- Enter

- Escape

- Arrow keys (Up, Down, Left, Right)

- Page Up

- Page Down

- Home

- End

### Editing Keys

- Backspace

- Delete

- Insert

- Space

### Function Keys

- F1-F12 (Note: Some may be reserved by the browser or OS)

### Safe Modifier Combinations

- Shift + \[any navigation key\]

- Shift + \[any editing key\]

- Ctrl/Cmd + A (Select All)

- Ctrl/Cmd + X (Cut)

- Ctrl/Cmd + C (Copy)

- Ctrl/Cmd + V (Paste)

- Ctrl/Cmd + Z (Undo)

- Ctrl/Cmd + Y (Redo)

## Browser/OS Considerations

### Reserved Combinations to Avoid

- Alt + Left Arrow (Browser back)

- Alt + Right Arrow (Browser forward)

- Ctrl/Cmd + N (New window)

- Ctrl/Cmd + T (New tab)

- Ctrl/Cmd + W (Close tab)

- Ctrl/Cmd + L (Focus address bar)

- Ctrl/Cmd + F (Find)

- Ctrl/Cmd + P (Print)

- Ctrl/Cmd + S (Save)

- Ctrl/Cmd + R (Reload)

- F5 (Refresh)

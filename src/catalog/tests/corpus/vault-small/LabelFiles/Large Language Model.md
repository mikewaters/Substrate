# Large Language Model

Alias:: LLM

---

Source: <https://twitter.com/fchollet/status/1755270681359716611>

Related: <https://fchollet.substack.com/p/how-i-think-about-llm-prompt-engineering>

> LLMs = 100% memorization. There is no other mechanism at work.
>
> A LLM is a curve fitted to a dataset (that is to say, a memory), with a sampling mechanism on top (which uses a RNG, so it can generate never-seen-before token sequences).
>
> 

> This is almost word for word David Hume's account of human cognition from A Treatise on Human Nature

# Info Augmentation

*Information Augmentation/Hydration*

Perform AI-based knowledge extraction from information datum, to be stored along with the capture,

Example: A reference to “Localfirst” might be difficult for me to recall (tip of my tongue…), but i might remember Inkbase or Ink&Switch. These terms are related, and so there will be a way to find a resource i am searching for by this type of matching.

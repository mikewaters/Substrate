A Component is some physical or digital entity that uses [Technology.md](./Technology.md) to provide access to some [Capability.md](./Capability.md)

A Component is some element described by a Concept that is part of a greater whole and has well-defined boundaries. 

A component, being an abstract concept, needs to have some implementation, and so that is one of its attributes

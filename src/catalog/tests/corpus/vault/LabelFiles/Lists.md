# Lists

A set of [Object (Data).md](./Object%20\(Data\).md)

- Lists of Project or Thought [Object (Data).md](./Object%20\(Data\).md): [Reference Store.md](./Reference%20Store.md)

   - Block can have structure, and these blocks contain URIs (location), and metadata including Ontology. This particular structure defines the block type [Reference (URI).md](./Reference%20\(URI\).md)

- Lists of Idea (etc) blocks, in [Mike’s Lists.md](./Mike’s%20Lists.md)

   - There can be overlap with Reference Store, in the case where one of these lists contains only DocumentReference blocks

- Directories in iCloud are just lists of [Document.md](./Document.md) blocks.

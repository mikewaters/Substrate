# Task and effort management with a time component

Given the efforts that are currently in scope, ensure we know exactly how to achieve them (tasks) and set yourself up to place those steps into real life.

### Near-term plans

- Weekly planning

- Seasonal planning

### Day to day task lists

#### Tasks by context

- Household chores

- Errands

   - Do I have to leave the house for something this week? Is there something that I need to find time to block for?

      Important data:

      - Due date

      - Blocked? Waiting?

- Call sheet/comms list

   - What communication is required in order to operate my life?

      Important data:

      - Due date

      - Follow-up status

      - Blocked?

      - Waiting?

   - Important calls

   - Text messages

   - Appointments to make

#### Tasks by priority

- Tasks with consequences

- Tasks with commitments

   - What tasks have I committed to another person, even implicitly?

      Important data:

      - Decay (how long has it been?)

- Tasks with deadlines

- Tasks with urgency or priority

#### Tasks by alignment

- Goal-based tasks

- Tasks related to a responsibility agreement

- Task possibilities that I enjoy

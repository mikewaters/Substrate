# Math Lessons

## “hat”

> In math, a caret (^) above a variable usually represents an estimate or approximation of that variable. For example, if you see \\(\\hat{x}\\), it often means that \\(x\\) is an estimated value, like in statistics where \\(\\hat{\\mu}\\) might be used to represent the estimated mean of a population.

It’s even called “hat” in latex

Is a:: Keyboard Shortcut on Beaker

Via:: Raycast

Language:: JXA/Javascript



Leverages a community [raycast script command called Copy Markdown URL and Title](https://github.com/chodorowicz/copy-markdown-url-and-title) (github)

[Blog post](https://www.chodorowicz.com/blog/copy-markdown-url-title)

Source code: `/Users/mike/Develop/Raycast/raycast-script-commands/copy-markdown-url-and-title.j`



I had previously attempted this, first with [a Safari extension](https://github.com/horimislime/safari-markdown-linker) (that I did not trust), then with a BetterTouchTool configuration (which didnt work), then using a HookMark shortcut (which was hard to use), and so the Raycast works fine.

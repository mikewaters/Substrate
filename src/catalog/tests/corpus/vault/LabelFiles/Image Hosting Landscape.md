# Image Hosting Landscape

Replacement for iPhoto, gPhoto etc

### Ente

Privacy focused, open source, hosting offering (5GB free)

<https://ente.io/>

<https://github.com/ente-io/ente>

e2e encryption verified with audit, local ML models for image analysis via [ONNX (Machine Learning Standard).md](./ONNX%20\(Machine%20Learning%20Standard\).md)

Hosted offering is replicated to three cloud providers, all in the EU.

Desktop and mobile apps for Apple and Google, based on Electron and Flutter/Dart, installable via homebrew

They have good docs <https://ente.io/reliability/>

## Cloud-based Services

### Amazon Comprehend

Amazon Comprehend now supports custom classification model training with native document formats like PDF, Word, and images\[7\]. Key features include:

- Layout support for improved accuracy

- Robustness in handling various document structures

- Reduced manual intervention

### Microsoft Syntex

A Microsoft 365 service that uses intelligent document processing and machine learning for document classification within SharePoint\[10\]. It offers:

- Pre-trained models for SharePoint

- Automated metadata extraction

- Integration with Microsoft PowerApps AI Builder

### Google Document AI

Google's AI document analysis tool can parse, split, and analyze unstructured and structured data from various document types\[10\]. Features include:

- Custom model creation with Document AI Workbench

- Document storage and search with Document AI Warehouse

- OCR capabilities for text digitization

## Specialized Classification Tools

### Zuva Document Classification

Zuva offers an extensive taxonomy of 220+ document types\[1\]. Benefits include:

- Prioritization of important documents

- Identification of documents requiring expert review

- Mass assignment and grouping of documents

### Mindee

Mindee provides automated document classification tools with features such as\[2\]:

- Customizable classification rules

- Integration with existing systems (ERP, CRM, CMS)

- Support for custom document types

### Parascript Document Classification

Parascript's software uses machine learning for document classification\[4\], offering:

- Automatic learning of key document features

- Visual element and digital text analysis

- Auto-discovery for unknown document types

## Open-Source and Lightweight Options

### SimpleIndex

SimpleIndex offers a simpler approach to document classification using keyword patterns\[5\]. Features include:

- Keyword-based classification

- Logical operators for precise matching

- Automatic workflow launching based on document type

### MiniLM and DistilBERT

These sub-7B models are efficient options for document categorization\[6\], providing:

- Solid performance with limited computational resources

- Suitability for applications with resource constraints

When choosing a document classifier service, consider factors such as accuracy requirements, integration needs, document volume, and specific industry requirements. Many services offer customizable solutions to meet unique classification needs.

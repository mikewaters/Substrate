# Automated Prioritization

One of:: [LifeOS Technical Capabilities.md](./LifeOS%20Technical%20Capabilities.md)

Using implicit and explicit importance, urgency, and relevance factors, prioritize captured datum and reflect that priority in a user-defined manner (tags, messages etc).

[Reference](https://miro.com/app/board/uXjVKFl5cpY=/?moveToWidget=3458764593754583075&cot=14)

1. Using the automated classification and matching, determine information importance:

   - matched an area of importance

   - matched an area of focus

   - matched an active effort

   - matched a high priority effort

2. Using signals from the user, determine priority

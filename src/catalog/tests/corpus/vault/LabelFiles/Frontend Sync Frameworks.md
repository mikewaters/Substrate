# Frontend Sync Frameworks

### Tanstack DB

Built on top of TanStack Query, which supports React, React Native, Solid, Svelte, Vuew, etc etc. Does not support any databases directly, at least as of Aug-2025.

Officially integrates with [Trailbase](https://app.heptabase.com/2f7caf87-d999-4778-8e30-61689601271e/card/35cde992-64ef-479e-a8d1-e7a234b24543#1837a353-6870-44ba-8692-b16d891cc0ff) and ElectricSQL

Does not support PGLite

#### TanStack Query

Officially integrates with [Trailbase](https://app.heptabase.com/2f7caf87-d999-4778-8e30-61689601271e/card/35cde992-64ef-479e-a8d1-e7a234b24543#1837a353-6870-44ba-8692-b16d891cc0ff) and ElectricSQL

#### Eval

Sister project, TanStack Query, [supports React Native](https://tanstack.com/query/v5/docs/framework/react/react-native). Perplexity says that DB doesnt, but I think it might be confused - it doesnt actually *use* a DB.
